OpenID Connect (OIDC)

OpenID Connect is an identity layer built on top of OAuth 2.0. While OAuth 2.0 is for authorization (granting access), OIDC is for authentication (verifying identity). It adds an ID Token with user information.

How CSRF Tokens Work:

  • Server generates unique token per session/request
  • Token included in forms as hidden field
  • Server validates token on submission
  • Evil sites can't access the token (Same-Origin Policy)

Additional CSRF Protections:

  • SameSite Cookies: Cookie only sent on same-site requests
  • Double Submit Cookie: Token in both cookie and form
  • Custom Headers: X-Requested-With for AJAX
  • Origin/Referer Validation: Check request source
